Battery Level Indicator

About

This Project will deterine the charge level of a 9 volt battery. The amount of charge will be relayed to the user through LED lights. If the battery is fully charged a green light will illuminate, yellow for partially charged and red if the battery is dead.

Information

Difficulty: Easy

Estimated time: 1-2 hours

Story

This useful little device will let you know when it is time to swap out you batteries. It is simple and fun for beginners to get into the fun of Arduino building and coding!

Components

The following hardware components were used in this project:

  • Arduino Uno
  • Small Breadboard)
  • 9 Male-to-female wires
  • 9 Volt battery Snap
  • 9 Volt battery
  • 3 LED's (green, yellow, red)
  • 3 330 ohm resistors
  • Arduino IDE

Instructions

Hardware connections:

There is very little wiring in this project. First, ground the breadboard by running a wire from the GRND port on the Arduino to the negative rail on the Breadboard. Next, wire the battery clip to the breadboard and run a wire from the same row as the batteries positive end(the red wire) to A0 on the Arduino Now you will put the LED lights on the bread board, and , wire the yellow light (through a resistor) to digital port 4, and wire the red light (through a resistor) to digital port 6. DO NOT forget to ground the LED lights with a wire from the negative rail to the negative end of the light(the shorter end)

   //R3 Battery Tester
#define newLED 2 //green  LED
#define okLED 4 // yellow LED
#define oldLED 6 //red LED

int analogValue = 0;
float voltage = 0;
int ledDelay = 3000;

void setup() {
  // put your setup code here, to run once:
  pinMode(newLED, OUTPUT);
  pinMode(okLED, OUTPUT);
  pinMode(oldLED, OUTPUT);

}

void loop() {
  // put your main code here, to run repeatedly:
  delay(1000);
  analogValue = analogRead(0);
  voltage = analogValue * 0.0048;
  if (voltage >= 1.6) {
    digitalWrite(newLED, HIGH);
    delay(ledDelay);
    digitalWrite(newLED, LOW);
  }
  else if (voltage >= 1.4) {
    digitalWrite(okLED, HIGH);
    delay(ledDelay);
    digitalWrite(okLED, LOW);
  }
  else {
    digitalWrite(oldLED, HIGH);
    delay(ledDelay);
    digitalWrite(oldLED, LOW);
  }

}
