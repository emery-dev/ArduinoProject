Light Show

About

This is final competition, you will be competing for eternal glory. For this project you will be using LED lights and Breadboards to create a light display set to the music of your choice. We will not be providing you a schematic or with code for this project as we expect you to be able to do this project without help and show off your creativity. Good luck and have fun!

Information

Difficulty: Medium

Estimated time: 2-3 hours

Story

You will be creating a light show in time with music, it can be as complex or as simple as you like, but remember this is a contest so show us what you got!

Components

The following hardware components were used in this project:

  • 1x Arduino Uno
  • LED Lights (Max 30)
  • 220 ohm Resistors (Max 30)
  • Wiring
  • Music!

Instructions

Overview:

For this project you will be mostly on your own, just remember that the longer wire in the LED is for the positive current, and the shorter wire is for the ground. Also, when connecting the LED to the positive current always use a resistor. If you do not remember how to code for the LED lights look back at the distance sensor project. Finally, get instructor approval for your choice of song early on, and make sure that you pick an appropriate song.
