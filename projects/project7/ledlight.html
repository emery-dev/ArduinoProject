Single LED Matrix

About

This tutorial will teach you how to control LED display units using the Arduino IDE and Arduino Uno board.

Information

Difficulty: Easy

Estimated time: 1 hour

Story

This project uses one single LED matrix and shows how to switch its LEDs on and off, and how to display ASCII characters. You can get very creative with an LED matrix- consider combining them, ways they may be useful to other projects, or even programming them differently.

Components

The following hardware components were used in this project:

  • Arduino IDE
  • Max7219 Red Dot Matrix Module
  • 5 Female-to-Male Jumper Wires

Instructions

Hardware:

The project is controlled from the MAX7219 chip through Arduino Library. Just like what you always do, install the library in your Arduino IDE: Sketch->Libraries->Add .ZIP Library If everything worked fine, you can now open the Arduino IDE again. If you go to File-> Examples, there should be a tab named LedControlMS with four files inside it. After installing the library, it’s time to connect the LED matrix to the Arduino board. The LED Matrix module has two sets of male header PINs. In this tutorial you will only use the input PIN headers that are located closer to the MAX 7219 chip.

Coding:

The library defines an alphabet/font of characters that can be displayed in the LED matrixes. It also includes two new methods, writeString() and displayChar(). You can see practical examples of how the different methods work in the MakeSpace_LEDMatrix example sketch distributed with the library.

If everything worked fine, your LED matrix will start a sequence of actions switching on and off leds and displaying different characters. You can now map the code in the example sketch to the actions in the matirx. Update the sketch to do new things and experiment with the library methods.

Code

Download the LEDControlMS.zip library.

To install the library, go to the menu "Sketch", then "Include Library", and then "Add .ZIP Library". Alternatively, go to "File", then "Examples", and then see if "LEDControlMS" is in the menu.
