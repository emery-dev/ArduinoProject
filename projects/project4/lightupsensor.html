Light Up Sensor

About

This project introduces the ultrasonic distance measuring sensor. You can use this sensor in many ways but this project will establish changing colors when the sensor reads input.

Information

Difficulty: Easy

Estimated time: 1 hour

Story

Utilizing a distance measurer can be useful for many Arduino projects. The main intention of this project is to help you understand how to use the ultrasonic distance sensor, and make you think about other projects that it might be implemented in. A few examples could be: a cookie-jar alarm with a buzzer, a distance measurer on a remote-controlled car, or even a door alarm. Below is a simple video of how the sensor should light up when input is read.

Components

The following hardware components were used in this project:

  • 1x Arduino Uno, or IEIK UNO
  • 1x Breadboard
  • 1x HC-SR04 Ultrasonic Distance Sensor
  • 2x Colored LED (any color)
  • 2x Resistors (560 ohm preferred, but any green, blue, brown, or gold resistor will do)
  • 8x Male-to-Male Wires

Hardware Setup:

Initialize the breadboard by setting up the positive and negative current flows. We connect the positive (+) to the "5v" and the negative (-) to the "GND". Continue setting up the breadboard and Arduino following the schematics above.

Coding Breakdown:

Setting up the Arduino is quite simple: 
void setup() {
  Serial.begin (9600);
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
  pinMode(led, OUTPUT);
  pinMode(led2, OUTPUT);
}

We define the pins as necessary to read the input from the ultrasonic distance sensor and also tell the Arduino how to handle the output (by lighting up the LEDs).

This is where the light change occurs. Calling pulseIn(echoPin, HIGH) returns the input from the sensor. We then check if the distance is close enough if (distance < 4) and then change the LED colors. See below for the full code implementation.

Code

#define trigPin 13
#define echoPin 12
#define led 11
#define led2 10

void setup() {
  Serial.begin (9600);
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
  pinMode(led, OUTPUT);
  pinMode(led2, OUTPUT);
}

void loop() {
  long duration, distance;
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);
  duration = pulseIn(echoPin, HIGH);
  distance = (duration/2) / 29.1;
    if (distance < 4) {
    digitalWrite(led,HIGH); 
    digitalWrite(led2,LOW);
    } else {
    digitalWrite(led,LOW);
    digitalWrite(led2,HIGH);
  }
  if (distance >= 200 || distance <= 0){
    Serial.println("Out of range");
  } else {
    Serial.print(distance);
    Serial.println(" cm");
  }
  delay(500);
}
